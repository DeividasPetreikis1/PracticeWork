This is work done on a US Presedent dataset in order to clean it up in excel.

The clean and unclean datasets can be found in this folder.

The cleaning that was conducted is:
- Removing duplicates
- Changing Column Names
- Changing Names To Same Format
- Fixing Errors
- Fixing Missing Spaces
- Removing Currency Symbol For SQL Use
- Fixing Dates
- Deleting Columns
